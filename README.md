# Лабораторная работа №1: Исследование типов данных в .NET Framework

## Описание программы

Консольное приложение на C# (.NET Framework 4.7.2) для анализа и исследования типов данных в среде .NET. Программа позволяет получить информацию о типах, встроенных в .NET Framework, а также проанализировать типы из загруженных сборок.

---

## Функциональные возможности

### 1. Общая информация по типам

- Общее количество загруженных сборок в домене приложения
- Общее количество типов во всех загруженных сборках
- Количество ссылочных типов (классов)
- Количество значимых типов
- Самое длинное название публичного свойства
- Тип с наибольшим количеством конструкторов
- Метод с наибольшим разнообразием типов аргументов

### 2. Информация о конкретном типе

Анализ следующих типов:

- `uint`, `int`, `long` — целочисленные типы
- `float`, `double` — типы с плавающей запятой
- `char` — символьный тип
- `string` — строковый тип
- `Tuple<int, string>` — обобщённый кортеж

Для выбранного типа отображается:

- Является ли тип значимым
- Пространство имён
- Имя сборки, в которой определён тип
- Общее число членов (методов, свойств, полей)
- Количество методов
- Количество свойств
- Количество полей
- Список полей
- Список свойств

> Дополнительно доступен вывод информации о методах с группировкой по типу возвращаемого значения.

### 3. Настройка консоли

- Изменение цвета шрифта (Чёрный, Белый, Красный, Зелёный, Синий)
- Изменение цвета фона (Чёрный, Белый, Красный, Зелёный, Синий)
- Защита от совпадения цвета шрифта и фона

## Классы и методы

### Класс `Program`

| Метод | Описание |
|-------|----------|
| `Main()` | Точка входа, главное меню |
| `ShowAllTypeInfo()` | Вывод общей информации по типам |
| `SelectType()` | Выбор типа для анализа из списка |
| `ShowTypeInfo(Type type)` | Вывод информации о выбранном типе |
| `ShowMethodInfo(Type type)` | Вывод информации о методах с группировкой |
| `ChangeConsoleView()` | Меню настроек консоли |
| `ChangeFontColor()` | Изменение цвета шрифта |
| `ChangeBackgroundColor()` | Изменение цвета фона |
| `GetAllTypes()` | Получение всех типов из загруженных сборок |

### Класс `TypeUtils`

Статический класс с утилитарными методами:

| Метод | Описание |
|-------|----------|
| `GetTheLongestPubProperty(List<Type> types)` | Возвращает свойство с самым длинным именем |
| `GetTypeWithMaxConstructors(List<Type> types)` | Возвращает тип с наибольшим количеством конструкторов |
| `GetMethodWithArgVariety(List<Type> types)` | Возвращает метод с наибольшим разнообразием типов параметров |

## Использование рефлексии

Программа активно использует механизм рефлексии (.NET Reflection) для анализа типов:

```csharp
// Пример получения всех методов типа
BindingFlags flags = BindingFlags.Public | BindingFlags.NonPublic | 
                     BindingFlags.Instance | BindingFlags.Static;
var methods = type.GetMethods(flags);

// Получение информации о конструкторах
var constructors = type.GetConstructors();

// Получение информации о свойствах
var properties = type.GetProperties();
```
